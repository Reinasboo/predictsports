## How to Get Listed

We recognize contributions of all kinds:
1. **Code submissions** - Submit a PR with meaningful changes
2. **Bug reports** - Open well-documented issues
3. **Feature suggestions** - Propose enhancements with clear use cases
4. **Documentation** - Improve README, guides, or code comments
5. **Testing** - Report issues found during testing
6. **Community** - Help answer questions and guide new contributors

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed contribution guidelines.
